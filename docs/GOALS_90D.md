# 岔路地图 · 90 天目标（2026-07-07 起）

主线确认：**B 为主 + A 为辅（70/30）**——先用产品跑通自己的真实闭环，路上顺势接创始价陪跑客户。

## 北极星（唯一成败判定）

> Phoebe 本人通过岔路地图完成的外联，在 90 天内拿到 1 个真实机会。

"真实机会"三选一即达成：

- 正式 offer
- 有转账的付费试点（金额不限）
- 签了明确范围的合作

口头夸奖、"保持联系"、无薪试用不算。

辅目标（被动接收，不主动推销）：2 个创始价陪跑客户（¥1,999，对外锚定原价 ¥4,999，创始价换案例授权）。

## 硬约束

- 每周总投入 **3 小时**，只花在"核实、发送、对话"三件人类动作上
- 预研草稿、内容初稿、产品开发、日志整理全部交给 AI 协作完成
- 产品开发不占用这 3 小时；功能只做主线直接需要的
- 所有对外材料过主张核查，诚实边界不因焦虑松动

## 每周模板（3 小时）

| 时间 | 动作 |
| --- | --- |
| 45 min | 确认 1 个目标对象（AI 预研 + 本人核实） |
| 40 min | 个性化并发出 1 封 |
| 15 min | 录入反馈 + 反馈校准 |
| 40 min | 发 1 篇公开过程帖（产品生成草稿，只改不写） |
| 40 min | 机动：回复、约聊准备；没有则不补进度 |

每周领先指标（只考核行动）：发送 ≥1、公开记录 ≥1、反馈 24h 内校准。

## 里程碑

**Day 30**

- 真实事实底稿入库（替换演示数据）
- 累计 4 个目标对象、3–4 封真实发送、3 篇公开记录
- 检查：发送 <3 → 问题在激活摩擦，修流程不修功能

**Day 60**

- 累计 8 封发送、≥1 个有效进展（约聊/试点讨论）
- 复盘页上线"诊断通话申请"入口，开始被动接收
- 检查：0 实质回复 → 停外联，用校准数据修目标画像和开口方式

**Day 90 判定**

- 达成北极星 → 切生意线重构（陪跑交付基建）
- 未达成但 ≥2 个有效进展 → 延长 30 天
- 全无进展 → 项目转纯作品集资产（体面结局），求职回归标准渠道并行

## 明确不做

- 账号体系 / 数据库 / 回复率数据平台（C 线，信号出现前不建）
- 主动招募陪跑客户的投放或私聊推销
- 每周 >1 天的产品开发
